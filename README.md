Implementation of the Graham scan using C, OpenGL and SDL2.

Linker options: 
  -lmingw32 -lSDL2main -lSDL2 -lopengl32

## Usage
  1) Left click to set point.
  2) Press "1" to see list of placed points.
  3) Press "2" to execute algorithm.
  4) "R" to reset.
  5) Thanks!
